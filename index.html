初始化本地Git环境

现在做点啥都需要Git,如果不知道什么是Git,请点击,简而言之,就是一个版本管理系统,并且是世界上的程序员都在使用.在本地设置环境,对于后面的操作都要方便很多.
安装过程依据选择的系统而不同,ubuntu下好像是最容易的,不过,这里要说的是安装之后设置本地环境的过程,打开命令界面..

  1. 查看版本
    • git –version/如果’–‘错误,试试’-‘
  2. 设置本设备用户名
    • git config –global user.name ‘kaoshanji-X550JX’(使用用户名-设备名…当使用多台电脑时,这也算预留)
  3. 设置邮箱
    • git config –global user.email ‘email@qq.com’
  4. 设置命令行
    • git config –global color.ui auto
  5. 生成秘钥
    • ssh-keygen -t rsa -C ‘email@qq.com’
  6. 在账户里添加
    • 登录:https://github.com/
    • 点击主页图标下’settings’,选择’SSH and GPG keys’
    • 点击’New SSK key’
    • 填写标题，复制生成的密钥目录下’id_rsa.pub’文件里的数据
    • 建议标题使用上面’设置本设备用户名’,这样,比较好对应
  7. 验证..
    • 输入 ssh -T git@github.com
    • 在上面填写的邮箱里会收到一封邮件
  8. 几个命令常用命令,一般这就是全部了..
    • 初始化:
      某目录Github/Atom 下 git clone git@…
    • 添加到暂存区:
      git add/rm (./加全部)
    • 提交至版本区:
      git commit -m ‘说明’
    • 推送服务器:
      git push
    • 拉取最新:
      git pull
